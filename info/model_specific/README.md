# Qwen3.5 Research Notes

This directory is a first-pass research pack for the `qwen/qwen3.5*` models that currently appear in
[`info/affordable_models_data_annotated.csv`](/Users/daniellerothermel/drotherm/repos/dr-llm/info/affordable_models_data_annotated.csv).

The goal is twofold:

1. Capture what the repo already knows about each model from the local CSV.
2. Point at the highest-value external sources for finding more evaluation data.

Source priority used here:

1. Official Qwen Hugging Face model cards
2. Official Qwen / Alibaba release posts and Model Studio docs
3. Existing repo-local annotated CSV values

The docs intentionally separate:

- `Already in our CSV`
- `Additional official benchmark reporting found`
- `Gaps / next searches`

That makes them usable both for manual study and for future annotation scripts.
