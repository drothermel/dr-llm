# Goal

Ingest, parse, normalize, standardize, and document the historical data around
encoding-decoding natural language and code.

The scope includes backups, pool dumps, database pool data, parsed reports,
generation logs, analysis artifacts, and other records from previous
experiments involving natural-language descriptions, code, and
encoder-decoder/direct generation workflows.

The desired end state is a clear inventory of source data plus standardized
artifact shapes that make the historical experiments comparable, auditable, and
usable for downstream analysis.
